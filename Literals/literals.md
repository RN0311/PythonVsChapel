## Literals
###### Python
```python
# Booleans
boolean1 = True
boolean2 = False

# Unsigned digits
unsigned_digits = 42 

# Signed digits
signed_digits = -42 

# Binary-Digits
binary_digits = 0b101010 

# Reals
real_number = 42.0 

# Strings
string1 = '42' 
string2 = "42"
```
###### Chapel
```Chapel
// Booleans
var boolean1 = true; 
boolean2 = false;

// Unsigned digits
var unsigned_digits = 42; 

// Signed digits
var signed_digits = -42; 

// Binary-Digits
var binary_digits = 0b101010;

// Reals
var r = 42.0; 

// Strings
var s = '42';

```
